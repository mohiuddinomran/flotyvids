Adds a floating video bubble to your WordPress site using a simple shortcode. Works with YouTube or self-hosted MP4.

== Description ==
Use this shortcode anywhere:

[floatyvids src="https://www.youtube.com/embed/VIDEO_ID"]

Or for a local video:

[floatyvids src="https://yoursite.com/wp-content/uploads/video.mp4"]

== Installation ==
1. Upload to /wp-content/plugins/
2. Activate plugin
3. Go to Settings > Floatyvids and enable options
4. Use shortcode on any post, page, or template
